# 海旅 - DivingPotal

このサイトは、スキューバダイビング愛好家のためのポータルサイトです。  
Instagramのダイビング投稿をハッシュタグで収集・表示し、観光情報やアフィリエイト連携も予定しています。

## 主な機能（予定・実装中）
- Instagramのハッシュタグ投稿の自動取得・表示
- エリア別のショップ一覧、ホテル情報、観光ガイド
- 利用者の体験談・写真投稿（今後追加予定）
